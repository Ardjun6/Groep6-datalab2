# Groep 6 – Datalab Sprint 2

## Over het project

Dit project is gemaakt voor **Datalab** door **Groep 6**.
In dit project analyseren we data van **FC Barcelona** met behulp van **Python en Jupyter Notebook** om inzicht te krijgen in prestaties en statistieken van het team.

## Projectstructuur

```
Groep6-datalab-sprint_2
│
├── assets/                     # Afbeeldingen, database(sqlite) en andere dingen komen hierin (als we deze repo verder gaan gebruiken voor sprint 3-6)
├── fc_barcelona_analysis.ipynb # Notebook voor sprint 2
├── README.md                   # Documentatie van het project
```

## Installatie

Clone de repository:

```bash
git clone https://github.com/Ardjun6/Groep6-datalab-sprint_2.git
cd Groep6-datalab-sprint_2
```

Installeer de benodigde libraries:

```bash
pip3 install pandas 
```
Open daarna:

```
fc_barcelona_analysis.ipynb
```
